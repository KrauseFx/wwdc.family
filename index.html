wwdc.family

Let's start a location sharing group for an exciting WWDC week in San Francisco

How does it work

Unfortunately Apple removed the group feature of the Find my Friends app.

As replacement we will use an iMessage group with location sharing enabled. You can opt-out any time.

Register

Fill out the Google Form and wait for further instructions.

Notes

Since we use an iMessage group, there will be potentially a lot of messages. I'd recommend muting the group.
